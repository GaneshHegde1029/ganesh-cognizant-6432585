# 🌐 Digital Nurture 4.0 – Java FSE Hands-On Assignments

This repository contains mandatory **hands-on exercises** for the **Digital Nurture 4.0 – Java Full Stack Engineer (FSE)** program. These tasks are designed to reinforce technical skills through real-world coding scenarios across the Java full stack.

## 📚 Topics Covered

- ✅ Spring Data JPA & Hibernate
- ✅ Spring Boot & REST API Development
- ✅ Microservices with Eureka Discovery
- ✅ PL/SQL Programming (Stored Procedures, Triggers)
- ✅ Testing with JUnit & Mockito
- ✅ DevOps Tools: Git, Docker (Basics)
- ✅ Basic Cloud & Generative AI Concepts

## 📅 Weekly Task Structure

Each week has specific coding assignments aligned with the training modules. Learners are expected to:

1. Clone or download the exercise content
2. Complete and test the hands-on tasks
3. Push the solution to their **personal GitHub** repository
4. Share the GitHub repo link as instructed for review

## 🚀 Objective

These exercises are crucial for:
- Building end-to-end Java full stack development skills
- Practicing project structure, RESTful design, and data handling
- Preparing for real-world problem-solving using modern Java technologies


## 📥 How to Use

1. Browse to the relevant week’s folder
2. Follow the instructions/files provided to complete tasks
3. Maintain clean code structure and commit history
4. Submit your GitHub repo link as proof of completion
